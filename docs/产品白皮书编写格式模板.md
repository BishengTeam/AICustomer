## 格式要点总结

| 要素 | 规范 |
|---|---|
| **标题层级** | `#` 产品名 → `##` 章 → `###` 节 → `####` 小节 |
| **截图占位** | 每章节/子功能均预留 `> 【截图位置】：XXX图/界面` |
| **强调样式** | 核心关键词、痛点、价值点使用 `**加粗**` |
| **表格** | 硬件配置、软件依赖用 Markdown 表格呈现 |
| **角色论述** | 多角色场景统一使用"对XX："的排比结构 |
| **结尾** | 统一附版权声明与迭代说明 |
