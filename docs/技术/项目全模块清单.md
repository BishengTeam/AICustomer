## 二、后端模块（Python FastAPI）

| 编号 | 模块名称 | 子模块/接口 | 功能说明 | 优先级 |
|------|----------|-------------|----------|--------|
| B-01 | **用户服务模块** | POST /api/auth/login、GET/PUT /api/user/profile | 微信登录(openid 换取)、用户信息 CRUD、手机号解密存储、身份类型/年龄采集 | P0 |
| B-02 | **AI 对话模块** | POST /api/chat、GET /api/chat/stream、GET /api/quick-questions | 消息接收、转发 Dify 工作流、流式 SSE 响应、意图识别路由、对话日志记录 | P0 |
| B-03 | **意图识别服务** | h3cne_intent_recognizer.py 扩展 | 认证类型识别、问题分类、风险等级判断、新增报名/价格/转人工意图输出 | P0 |
| B-04 | **报名缴费模块** | POST /api/orders、POST /api/payment/prepay、GET /api/orders/{id}、GET /api/orders | 订单创建、微信支付统一下单、支付回调处理(幂等)、订单状态机(待支付→已支付→已完成/已退款)、订阅消息推送 | P0 |
| B-05 | **价格配置模块** | PRICE_CONFIG 表 CRUD | exam_type + user_type → price 映射，企业员工¥1200/学生¥600，后台可调 | P0 |
| B-06 | **人工客服转接模块** | POST /api/tickets、会话状态管理 | 转人工触发、会话标记(waiting_manual/in_manual_chat/resolved)、老师分配、工单创建 | P1 |
| B-07 | **认证列表服务** | GET /api/certifications | 返回考试类型列表(名称/描述/价格/图标)，供报名专区页使用 | P0 |
| B-08 | **专区内容服务** | GET /api/zones (聚合页)、GET /api/zones/exam (考试专区)、GET /api/zones/study (学习专区)、GET /api/zones/competition (竞赛专区)、GET /api/zones/activity (活动专区)、GET /api/zones/employment (就业专区)、CRUD /api/admin/zones (后台管理) | 聚合页专区列表、5个子页各自内容列表(图文/外链/跳转配置)、后台内容增删改排序上下线 | P1 |
| B-09 | **微信生态对接模块** | wx.login code2session、getPhoneNumber 解密、subscribeMessage.send、wxpay 统一下单/回调 | 微信登录、手机号获取、订阅消息推送、微信支付全链路 | P0 |
| B-10 | **输入安全审查模块** | 前置中间件 | 敏感词过滤、Prompt 注入检测、违规内容拦截 | P0 |
| B-11 | **输出合规校验模块** | 后置中间件 | 内容合规校验(政治/色情/暴力/违规承诺)、高风险字段与官方口径库比对 | P0 |
| B-12 | **高风险字段校验模块** | HIGH_RISK_RULE 表比对 | 考试代码/有效期/重认证规则等高风险内容最终口径校验 | P1 |
| B-13 | **微信小店/课程引流模块** | 商品卡片数据组装 | AI 对话中识别学习意图后，返回微信小店商品卡片或 H5 学习网站链接 | P2 |
| B-14 | **RAG 知识库层** | Dify Workflow + H3CNE 切片 | 向量检索、父子上下文检索、查询重写、知识来源回溯(Citation) | P0(已有) |
| B-15 | **数据统计分析模块** | 报名转化率、AI 问题解决率、转人工率、对话日志查询 | 运营后台数据接口支撑 | P2 |
| B-16 | **管理后台 API** | 对话管理/订单管理/用户管理/内容配置/数据分析全维度 CRUD | 管理后台 Web 端的全部后端接口 | P2 |
