# H3CNE 测试问题清单 V1

## 1. 使用说明

本文件用于测试 Dify 中 H3CNE AI 客服的回答效果。

测试目标：

- 验证是否能命中正确知识
- 验证是否能做澄清
- 验证是否能拒答
- 验证答案是否带来源且不乱编

## 8. 测试结论

- 当前主要问题：考试代码仍无法命中；重认证问题仍缺少可直接回答的知识；报名入口仍偏泛化、缺少明确 URL 或固定渠道名；来源仍不够可追溯。
- 当前可演示场景：项目介绍、适合人群、成绩查询、证书下载、证书有效期、学习建议、培训非强制、口语化报名/有效期识别。
- 必须优先修复的问题：补齐考试代码与重认证的高风险正式知识；把报名入口改成“明确平台/路径/步骤”；为高风险回答补可审计来源。
