# H3CNE-RS+ 清洗数据说明

## 1. 目录用途

本目录用于存放基于 `docs/H3CNE` 现有资料清洗后的 **H3CNE-RS+ 知识库导入版文档**。

清洗目标：

- 统一主题为 **H3CNE-RS+**
- 去掉流程文档、测试文档、风险文档中的内部措辞
- 去掉明显冲突或待核验字段
- 按知识主题拆分，便于 Dify 知识库导入

---

- 高风险字段在未核验官方页面前不应直接进入知识库

---

## 4. 当前清洗原则

本轮清洗采用以下原则：

- 只保留本地资料中语义稳定、相互不冲突的内容
- 对考试代码、是否强制培训、报名平台细节等高风险字段采取保守策略
- 统一使用 `H3CNE-RS+` 作为当前 MVP 的知识主题
- 不混入“测试目标、工作流、待确认、建议拒答”等内部内容
