# H3CNE-RS+ 报名方式、成绩查询与证书发放

## 1. 报名方式

根据当前本地资料，H3CNE-RS+ 认证考试可通过以下渠道进行预约或报名：

1. ATAC 考点预约考试
2. 新华三技术认证小程序进行缴费并与考点预约考试
3. Prometric 官网查询考点并预约考试

现有资料同时提到，H3CNE-RS+ 认证考试由 ATAC 和 Prometric 考试平台代理。

---

## 2. 报名说明

现有资料更适合支撑以下结论：

- H3CNE-RS+ 考试属于预约考试模式
- 用户可以通过官方或授权渠道完成报名与预约
- 小程序与考点预约是当前资料中反复出现的主要渠道

由于本地资料尚未完成对报名入口页面的逐项核验，知识库在回答时更适合使用“可通过官方小程序、授权考点或考试平台预约考试”的表述，不宜扩展到更细的流程和费用细节。

---

## 3. 成绩查询

根据本地资料，已获取的证书及相关考试信息可通过以下两种方式查询：

### PC 端

点击“证书查询与下载”链接，输入账号和密码，登录后可查询考试成绩及证书信息。

当前资料中该入口对应 URL 为：

`http://www.h3c.com/CN/BizPortal/TrainingPartner/Tester/AP_Edu_SearchScoreLogin.aspx`

### 移动端

访问“新华三技术认证官微 >> 技术认证 >> 认证考试及证书服务 >> 证书查询”，输入账号和密码，登录后可查询考试成绩及证书信息。

---

## 4. 证书发放

根据当前资料：

- 查询时使用的账号和密码会随着证书发放通知邮件一并发放
- 证书发放时间为考试通过后一周内
- 未通过考试的考生无法查询到成绩
- 如果用户问“拿证入口 / 领证入口 / 证书下载入口 / 电子证书在哪领”，可回答 PC 端“证书查询与下载”入口及移动端“新华三技术认证官微 → 技术认证 → 认证考试及证书服务 → 证书查询”路径

由于该部分属于时效性较强的信息，知识库正式使用前仍建议核验一次官方页面。
