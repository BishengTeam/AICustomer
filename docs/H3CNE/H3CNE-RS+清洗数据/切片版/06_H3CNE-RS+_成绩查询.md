# H3CNE-RS+ 成绩查询
问：H3CNE-RS+ 成绩在哪里查询？  
答：按当前资料，PC 端可通过“证书查询与下载”入口查询，网址为 `http://www.h3c.com/CN/BizPortal/TrainingPartner/Tester/AP_Edu_SearchScoreLogin.aspx`；移动端可通过“新华三技术认证官微 → 技术认证 → 认证考试及证书服务 → 证书查询”进入。该入口用于查询考试成绩及证书信息，不应理解为只能查成绩。

问：H3CNE-RS+ 成绩查询需要什么信息？  
答：当前资料显示，查询需要 `账号 + 密码`。

问：H3CNE-RS+ 移动端怎么查成绩？  
答：可通过“新华三技术认证官微 → 技术认证 → 认证考试及证书服务 → 证书查询”进入，登录后查询考试成绩及证书信息。

问：这个“证书查询与下载”入口能不能用于证书信息查询？  
答：可以。当前资料表述为登录后可查询考试成绩及证书信息；如果用户问“证书在哪里查 / 拿证入口 / 证书下载入口”，也可以引用该 PC 端入口和移动端官微路径，但要提示具体页面以官方最新展示为准。

## PC 端查询方式补充

根据当前资料，用户可通过“证书查询与下载”入口查询考试成绩及证书信息：
