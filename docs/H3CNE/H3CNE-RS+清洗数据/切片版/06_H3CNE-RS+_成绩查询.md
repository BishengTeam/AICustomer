# H3CNE-RS+ 成绩查询

## 一句话结论

当前资料支持这样回答：

- PC 端可通过“证书查询与下载”入口查询成绩
- 入口 URL：`http://www.h3c.com/CN/BizPortal/TrainingPartner/Tester/AP_Edu_SearchScoreLogin.aspx`
- 移动端可通过“新华三技术认证官微 -> 技术认证 -> 认证考试及证书服务 -> 证书查询”进入
- 查询需要：`账号 + 密码`

---

## PC 端查询方式

根据当前资料，用户可通过“证书查询与下载”入口查询考试成绩及证书信息：

- URL：`http://www.h3c.com/CN/BizPortal/TrainingPartner/Tester/AP_Edu_SearchScoreLogin.aspx`
- 登录方式：输入账号和密码
- 登录后可查询：考试成绩及证书信息

---

## 移动端查询方式

根据当前资料，用户可通过“新华三技术认证官微”进入：

技术认证 → 认证考试及证书服务 → 证书查询

登录后可查询考试成绩及证书信息。
