# H3CNE-RS+ 证书有效期

## 一句话结论

`H3CNE-RS+` 证书有效期为 `3年`。

如果用户只问“证书多久过期 / 有效期几年”，优先先回答：

- `证书有效期为 3 年。`

不要先展开重认证或刷新有效期规则，除非用户继续追问。

---

## 有效期说明

根据当前资料，考生通过 H3CNE-RS+ 考试后，将获得由 H3C 统一签发的 H3CNE-RS+ 证书，证书有效期为 3 年。

---

## 直接回答模板

可直接用于回答：

- `H3CNE-RS+ 证书自签发之日起有效期为 3 年。`
