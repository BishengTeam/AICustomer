# H3CNE-RS+ Dify 切片版说明

## 1. 目录用途

本目录用于存放更适合 Dify 知识库切片的 `H3CNE-RS+` 文档版本。

设计原则：

- 一份文档只讲一个主题
- 标题明确，便于按标题切片
- 内容短而稳定，减少跨主题混淆
- 高风险字段尽量保守表达

---

## 2. 导入建议

建议直接导入本目录下的 `01` 到 `09` 文件。
高风险字段模板已移动到 `src/Suggestion/10_H3CNE-RS+_高风险字段正式版.md`，补齐核验信息后再导入知识库。

推荐导入设置：

- 按标题切片
- 保留标题
- 切片长度适中
- 命中后返回 3~5 个片段

---

## 3. 暂不纳入

本目录不包含以下内容：

- 工作流说明
- 测试问题
- 风险清单
- 待核验冲突字段汇总

这些内容不适合直接进入知识库。
