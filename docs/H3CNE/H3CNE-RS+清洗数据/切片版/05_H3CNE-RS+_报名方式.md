# H3CNE-RS+ 报名方式
问：H3CNE-RS+ 去哪里报名？  
答：按当前资料，H3CNE-RS+ 可先从 `https://c.h3c.com/cn/` 进入认证相关入口页，再通过 `新华三技术认证小程序` 或授权考试渠道（`Prometric` / `Pearson VUE` / `ATAC`）预约考试。

问：H3CNE-RS+ 怎么报名？  
答：建议按“进入认证入口页 → 选择考试渠道 → 注册账号 → 预约考试 / 缴费”的顺序操作。

问：H3CNE-RS+ 只能通过授权培训中心报名吗？  
答：不是。当前资料支持的常用渠道包括 `https://c.h3c.com/cn/`、`新华三技术认证小程序`、`Prometric`、`Pearson VUE` 和 `ATAC`；回答时不要只收缩成“联系授权培训中心”。

---

## 操作步骤补充

### 方式一：从官方入口进入

1. 打开 `https://c.h3c.com/cn/`
2. 进入 H3C 认证相关服务页面
3. 选择对应考试或预约渠道（如小程序、 Prometric、Pearson VUE、ATAC）
4. 按页面提示完成账号注册、约考与缴费

### 方式二：通过常用预约渠道

可按当前资料使用以下渠道：

1. `新华三技术认证小程序`
2. `Prometric`
3. `Pearson VUE`
4. `ATAC`

---

## 使用边界

报名问题回答时，建议优先说清楚这 3 点：

1. `去哪里报`：优先先说 `https://c.h3c.com/cn/`
2. `通过什么报`：小程序 / Prometric / Pearson VUE / ATAC
3. `怎么做`：注册账号 → 选择考试 → 预约时间 / 考点 → 缴费

---

## 使用提醒

- 报名平台和可用渠道属于高风险信息，页面可能调整
- 正式回答时建议补一句“具体入口与可用性请以官方最新页面实时展示为准”
- 不要只回答“联系授权培训中心”或“去官网看”，尽量同时给出**入口 URL**、**渠道名**与**操作步骤**
