# H3CNE-RS+ 认证简介
问：H3CNE-RS+ 是什么？  
答：H3CNE-RS+ 是新华三（H3C）认证体系中面向路由交换方向的入门级技术认证。

问：H3CNE-RS+ 是干嘛的？  
答：它主要面向中小型网络的规划、设计、配置与维护场景，用于证明学习者具备网络技术入门阶段的基础能力。

问：H3CNE-RS+ 是不是入门级认证？  
答：是。按当前资料，H3CNE-RS+ 属于网络技术入门阶段常见的基础认证。

---

## 能力范围补充

根据当前资料，H3CNE-RS+ 主要用于证明考生具备以下基础能力：

- 数据通信网络基础理论理解能力
- H3C 网络设备基础操作能力
- 中小型网络部署与维护能力
- 基础网络故障排查能力
