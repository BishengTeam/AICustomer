# H3CNE-RS+ 证书发放与下载
问：H3CNE-RS+ 证书怎么领取或下载？  
答：按当前资料，用户可通过 PC 端“证书查询与下载”入口或移动端认证服务入口，登录后查询并下载证书信息。PC 端入口可使用 `http://www.h3c.com/CN/BizPortal/TrainingPartner/Tester/AP_Edu_SearchScoreLogin.aspx`；移动端可通过“新华三技术认证官微 → 技术认证 → 认证考试及证书服务 → 证书查询”进入。

问：H3CNE 拿证入口在哪？  
答：当前专题默认按 H3CNE-RS+ 理解。拿证 / 证书查询 / 证书下载可优先回答两个入口：1）PC 端“证书查询与下载”入口 `http://www.h3c.com/CN/BizPortal/TrainingPartner/Tester/AP_Edu_SearchScoreLogin.aspx`；2）移动端“新华三技术认证官微 → 技术认证 → 认证考试及证书服务 → 证书查询”。登录后可查询考试成绩及证书信息。具体入口以官方最新页面为准。

问：H3CNE 电子证书在哪领？  
答：通过考试后，可通过 PC 端“证书查询与下载”入口或移动端“新华三技术认证官微”的证书查询路径查看证书信息并下载电子证书。查询时通常需要账号和密码。

问：查询证书时用什么账号信息？  
答：当前资料提到，查询时使用的账号和密码会随着证书发放通知邮件一并发放，用户需注意保存相关信息。

问：H3CNE-RS+ 证书多久发放？  
答：根据当前资料，证书发放时间为考试通过后一周内；该时效属于较强时效性信息，正式对外回答时建议补一句“以最新官方通知为准”。

- 证书发放与下载属于查询入口类问题，回答时尽量同时说清楚“去哪里查”和“用什么账号查”
- 用户问“拿证入口 / 领证入口 / 证书下载入口 / 电子证书在哪领”时，应优先命中本切片并给出 PC 端入口、移动端官微路径和登录信息
- 不要只回答“当前资料无法确认”；当前资料已经给出“证书查询与下载”入口及移动端查询路径
- 证书发放时效属于较强时效性信息，正式对外使用时建议补充“以最新官方通知为准”
