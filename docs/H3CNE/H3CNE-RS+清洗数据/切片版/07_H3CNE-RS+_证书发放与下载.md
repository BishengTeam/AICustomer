# H3CNE-RS+ 证书发放与下载
问：H3CNE-RS+ 证书怎么领取或下载？  
答：按当前资料，用户可通过 PC 端“证书查询与下载”入口或移动端认证服务入口，登录后查询并下载证书信息。

问：查询证书时用什么账号信息？  
答：当前资料提到，查询时使用的账号和密码会随着证书发放通知邮件一并发放，用户需注意保存相关信息。

问：H3CNE-RS+ 证书多久发放？  
答：根据当前资料，证书发放时间为考试通过后一周内；该时效属于较强时效性信息，正式对外回答时建议补一句“以最新官方通知为准”。

## 使用提醒

- 证书发放与下载属于查询入口类问题，回答时尽量同时说清楚“去哪里查”和“用什么账号查”
- 证书发放时效属于较强时效性信息，正式对外使用时建议补充“以最新官方通知为准”
