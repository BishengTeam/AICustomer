## 3. 需优先澄清的问题

以下问题不应直接回答，建议先追问：

| 问题 | 为什么要澄清 | 建议澄清方式 |
| --- | --- | --- |
| 我适合考哪个证？ | 缺少基础、岗位、方向信息 | 询问基础、岗位、目标方向 |
| 我这个能不能报？ | 缺少具体认证项目 | 询问是否指 H3CNE |
| 这个证还有用吗？ | 可能问价值，也可能问有效期 | 询问是证书价值还是有效期 |
| 这个怎么报名？ | 可能问官方入口，也可能问甲方服务 | 询问是官方报名还是培训报名 |

## 4. 建议直接拒答的问题

| 问题 | 原因 | 建议答法 |
| --- | --- | --- |
| 你保证我一次过吗？ | 无法承诺结果 | 明确不能保证考试结果 |
| 你告诉我本次真题吧 | 不合适回答 | 明确拒绝 |
| 直接给我答案题库 | 不合适回答 | 明确拒绝 |
| 我花多少钱你能帮我搞定证书？ | 高风险/不合规 | 明确拒绝 |

## 6. 给 B 的配置建议

- 考试代码

- 成绩时效
- 证书有效期

### 建议可直接回答的问题域

- 项目介绍
- 适合人群
- 一般性学习路径
