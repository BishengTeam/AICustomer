# H3CNE-RS+信息收集

##  H3CNE认证项目介绍

H3CNE（H3C Certified Network Engineer，H3C 认证网络工程师）是新华三（H3C）推出的入门级技术认证，核心定位于中小型网络的规划、设计、配置与维护，证明持证者掌握数据通信网络基础理论与 H3C 设备实操能力，是网络技术入行的 “敲门砖”。

## H3CNE-RS+认证适用的对象包括：
1.中小企业网络工程师和IT管理人员
2.本科、高职院校的计算机类或电子信息类专业学生
3.IT技术爱好者
4.H3C公司代理商工程师
5.H3C公司培训合作伙伴教师
6.H3C公司网络产品维护和技术支持人员

## 培训课程

1. H3CNE-RS+培训课程以官方指定教材《路由交换技术基础 V1.0》为核心，采用理论+实验的5 天精品面授或标准化在线课程体系，适配零基础入门与岗位技能提升需求。
2.课程培训内容主要有：课程内容主要涵盖计算机网络基础、H3C网络设备操作入门、配置局域网交换、高级TCP/IP知识、配置IP路由、配置安全的分支网络、广域网接入和互连、SDN与网络虚拟化等方面。

## 考试代码

获得证书需要通过以下考试，考试代码为：GB0-192

## 报名方式
H3CNE-RS+认证考试由ATAC和Prometric考试平台代理。
报名方式1.在H3C新华三官网点击'考点查询'可以查询ATAC考点预约考试。
报名方式2.通过‘新华三技术认证小程序’进行缴费并与考点预约考试。
报名方式3.登录Prometric官网查询考点预约考试。

## 考试成绩查询
已获取的证书可通过以下两种方式查询和下载：
1.PC端：点击"证书查询与下载"链接，输入账号和密码，登录后可查询考试成绩及证书信息。
2.移动端：访问“新华三技术认证官微 >> 技术认证 >> 认证考试及证书服务 >> 证书查询”，输入账号和密码，登录后可查询考试成绩及证书信息。

## 证书发放
查询时使用的账号和密码会随着证书发放通知邮件一并发放，请注意保存相关信息。证书发放时间为考试通过后一周内，未通过考试的考生无法查询到成绩。

## 证书有效期和重认证
考生通过H3CNE-RS+考试后，将会获得由H3C公司统一签发H3CNE-RS+证书，证书有效期3年。有如下方式可以刷新证书有效期，任选其一即可。
1.证书到期之前，通过有技术相关性的上级认证（包括H3CSE-RS+、H3CSE-SDN、H3CSE-WLAN、H3CIE-RS+、H3CIE-SDN、H3CIE-WLAN）中的任一认证的全部考试，都可以刷新H3CNE-RS+的有效期。
2.证书到期之后，考生需要重新通过H3CNE-RS+认证的相应考试以获得H3CNE-RS+证书，以便考取更高级别的证书。
