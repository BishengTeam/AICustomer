### 共性问题
1. 内部标注信息混入RAG切片：大量面向运营/标注人员的内部提示、使用规则、检索触发词、回答边界等非知识内容出现在切片中，属于严重噪声
2. 内容冗余：同一知识点重复出现在多个切片，同一切片内存在重复表述
3. 切片编号混乱：存在跳号、顺序颠倒问题，不利于管理
4. 时效性提示不规范：时效性强的内容提示位置不统一，格式不一致

### 改进规则
**Why:** 这些问题会影响RAG检索准确性和回答质量，不符合知识切片的质量标准
**How to apply:** 对所有RAG切片执行以下检查：
1. 必须剥离所有内部运营标注内容，仅保留面向用户的问题、回答和知识点
2. 合并重复内容，明确每个切片的单一主题边界
3. 切片编号必须连续、顺序合理
4. 时效性内容统一在回答末尾添加"具体请以官方最新公告为准"的提示
